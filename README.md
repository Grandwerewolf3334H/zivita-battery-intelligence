

# Zivita Battery Intelligence

An AI-driven battery analytics platform for second-life lithium-ion battery applications.

## Project Objective

Develop machine learning models to:

- Estimate State of Health (SOH)
- Grade retired battery cells
- Support second-life battery reuse
- Enable future Remaining Useful Life (RUL) prediction

## Dataset

NASA Battery Aging Dataset

Features extracted:

- Capacity
- Voltage
- Current
- Temperature
- Discharge Duration
- Electrolyte Resistance (Re)
- Charge Transfer Resistance (Rct)

## Current Results

SOH Prediction Model

Model:
Random Forest Regressor

Performance:

- MAE = 0.816
- R² = 0.968
